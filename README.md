# Шаблон Легковаговик (Flyweight, Python)

Цей проєкт демонструє шаблон проєктування **Flyweight** на Python.

Flyweight дозволяє економити пам’ять, використовуючи спільні об’єкти для 
повторно використовуваних даних. Він особливо корисний, коли потрібно 
створити велику кількість об’єктів із однаковими внутрішніми даними.

## Структура
- flyweights/ – клас спільного стану (Flyweight)
- forest/ – колекція об’єктів, яка управляє Flyweight
- main.py – клієнтський код

## Використання
Симуляція лісу, де багато дерев одного типу:
- TreeType (спільні дані: колір, тип, текстура)
- Tree (унікальні координати x, y)
